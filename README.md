# vpn-anyconnect-client
思科 anyconnect 客户端 、账户 翻墙方法 ios、安卓、win 、maxos
## 安装方法
### ios系统：
打开移动端app商店搜索“anyconnect” 第一个软件就是，下载并安装
### 安卓用户：
下载 思科安装包文件夹下的apk文件，自行在移动端安装
### win
下载 思科安装包文件夹下的 msi文件，打开安装程序安装
### macos 
下载 思科安装包文件夹下的 dmg文件，进入安装程序，点击同意，自行安装
